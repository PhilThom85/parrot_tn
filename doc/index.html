ParrotTn

This utility gem enables to translate from local dictionnaries and/or google translator.

Another feature is to translate a yml file, usually an i18n file, into another yaml file with the same hierarchy.

Installation

Add in Gemfile

gem “parrot_tn”, :git => “git://github.com/PhilThom85/parrot_tn.git”

Usage

Command line

In order to translate a text with the help of google translate :

bundle exec parrot “My text to translate” en:fr

where: en is source language and fr is destination language

Ruby code

Standard use

This is used to translate a text to another with or without google translation

#… guess = ParrotTn::Guess.new :google => true puts guess.translation_of “My text to translate”, :from => :en, :to => :fr #…

Convert yaml file

To convert a yaml file, use :

parser = ParrotTn::ParserYml.new :google => true parser.parrotize_yaml “path_to_file”, :fr

The resulting file is saved to the current directory but can be changed using :save_to option. The source file must start with the code language as root, as shown below:

en: …

Usually the file is an i18n file :

en: code_a: “House” code_b: – “Monday” – “Tuesday”

will be converted to french for instance, as shown bellow:

fr: code_a: “Maison” code_b: – “Lundi” – “Mardi”
